# OS Codework Landing Page

A modern, responsive landing page developed as part of an assignment for **OS Codework**. The project focuses on creating a visually appealing, user-friendly interface with clean layouts, responsive design, and intuitive navigation. It demonstrates strong front-end development skills and attention to user experience.

## 🌐 Live Demo

https://sakhareprajakta.github.io/os-codework/

## 📖 About

The OS Codework Landing Page was created to showcase modern web design principles and responsive front-end development. The objective was to build an engaging and professional website that provides users with a seamless browsing experience across all devices.

The design emphasizes clean typography, balanced spacing, attractive visual hierarchy, and responsive layouts to ensure accessibility and usability. Every section is structured to improve readability while maintaining a modern and elegant appearance.

This project demonstrates proficiency in building responsive websites using semantic HTML, CSS, and JavaScript while following best practices in UI design and front-end development.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/sakhareprajakta/os-codework.git
```

### Navigate to the Project

```bash
cd os-codework
```

### Run the Project

Simply open `index.html` in your preferred web browser.

Or use **Live Server** in Visual Studio Code:

1. Install the Live Server extension.
2. Right-click on `index.html`.
3. Select **Open with Live Server**.
